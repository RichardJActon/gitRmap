Quick Start:

To add additional locations to the map:

  • Go to https://github.com/nickbearman/gitRmap/
  • Fork: Click Fork, to create a fork
  • Edit the file data/locations.csv
    • Make your changes, Click Commit Changes
  • Open a pull request

Made with: gitRmap / "Guitar Map"

gitRmap is a semi-automated map of locations, which you can add your own points to. Designed for those who are new to GIS, but know a little bit of Git.

Create a map like this, with your own pins on it, by following the quick start below.
